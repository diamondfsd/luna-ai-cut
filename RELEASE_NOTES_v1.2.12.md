# 发布说明 v1.2.12

## 新功能

- **视频导出功能** — 新增 ffmpeg pipeline 模块化架构，支持视频导出时：
  - 叠加自定义水印（位置、大小、样式可调）
  - 调整输出分辨率、帧率、码率
  - 支持转码、水印预览等完整流程
- **导出队列预览列表** — 导出进度弹窗增加队列预览列表，直观查看所有待导出任务的进度状态
- **导出示意图功能** — 支持导出设置中的图片可拖拽可视化调整布局
- **新增 UI 共享组件**：
  - `Accordion` 手风琴折叠面板，用于导出设置中的参数分组
  - `Select` 下拉选择器，替代原生 `<select>` 实现统一风格

## Bug 修复

- **导出进度修复** — 导出进度改用 `exportId` 做 Map key，解决同一文件多次导出时进度记录互相覆盖的问题
- **遮罩层无法关闭面板** — 修复 DropdownPanel 和 ModalLayer 中点击遮罩层无法关闭面板的问题
- **Windows 构建 ffmpeg 缺失** — 修复在 macOS 上为 Windows 交叉编译打包时，`ffmpeg.exe` 未包含在安装包中的问题（新增 `--target` / `--arch` 参数支持且含下载缓存机制）

## UI 变化

- **导出设置布局调整** — 导出弹窗布局优化，水印设置项分组更清晰
- **调整设置面板** — 导出参数设置布局重构，更好地适配各项参数配置

## 重构

- **ffmpeg 模块化重构** — 将水印处理逻辑从单一 `watermarkService.ts` 拆分为可插拔的 pipeline 模块架构（`pipeline.ts` / `bitrate.ts` / `codec.ts` / `framerate.ts` / `scale.ts` / `watermark.ts`），各模块职责单一、可独立复用
- **构建脚本优化** — `copy-ffmpeg.mjs` 新增交叉编译支持、代理检测和缓存机制，提升构建效率和可靠性
